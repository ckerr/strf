[%header,cols="382,618"]
|===
| Member function | Effect

|`operator<({width_t} width)`
| Aligns to the left ( Or to the right right-to-left (RTL) script )

|`operator>({width_t} width)`
| Aligns to the right ( Or to the left right-to-left (RTL) script )

|`operator^({width_t} width)`
| Center alignment

|`fill(char32_t ch)`
| Sets the fill character.
|===
